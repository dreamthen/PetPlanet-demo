# 逼疯的铲屎官

PS: 需要node > 8的node版本

> 搭建

```bash
    npm install -g @tarojs/cli
```

```bash
    npm install 
    #或者
    npm i
    #或者
    yarn
```

> 开发环境

```bash
    yarn run dev:weapp
    #或者
    npm run dev:weapp
```

> 生产环境

直接在微信开发者工具里面,点击上传即可
